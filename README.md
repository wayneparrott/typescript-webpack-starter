# Typescript Webpack Starter

- Typescript support.
- Webpack
- Html plugin
- BrowserSync

## Usage

### Install Typscript

```
npm install typescript -g
```

### Start

```
$ git clone https://github.com/wayneparrott/typescript-webpack-starter.git
$ cd typescript-webpack-starter
$ npm install # or yarn
$ npm start
```
